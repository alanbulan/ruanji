# 更新日志

所有重要的版本更新都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2026-01-30

### 新增
- 软件列表功能：扫描系统已安装软件，显示名称、版本、大小、分类
- 软件分类：支持 50+ 类别的智能分类（游戏、办公、开发工具等）
- 软件迁移：将软件从 C 盘迁移到其他磁盘，自动创建符号链接
- 文件管理：文件浏览、搜索、类型统计、大小分析
- AI 文件分析：集成 OpenAI 兼容接口进行智能文件分析
- 系统清理：扫描并清理软件残留文件
- 设置页面：OpenAI API 配置，支持动态加载模型列表

### 技术特性
- 基于 WinUI 3 (Windows App SDK 1.5) 构建
- 使用 .NET 8 和 C# 12
- MVVM 架构模式
- 支持 Windows 10 19041+ 和 Windows 11
